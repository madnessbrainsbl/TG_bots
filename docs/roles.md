# Роли и инструкции

## 👤 Партнёр
- Приглашает новых участников.
- Смотрит свою статистику продаж и бонусов.
- Может оставить отзыв.
- Получает уведомления о новых материалах, новостях и новых партнёрах в структуре.

## 🛠 Админ
- Полный доступ к системе.
- Управление ролями пользователей.
- Массовые рассылки уведомлений.
- Модерация отзывов (одобрить/отклонить).
- Настройка глубины MLM и процентов.

## 📰 Контент-менеджер
- Добавляет/редактирует материалы для приглашения.
- Публикует новости.

## 👀 Модератор
- Проверяет новых пользователей.
- Модерирует отзывы.
- Контролирует корректность данных.

## 📊 Функции статистики
- Пользователь: личная статистика и структура.
- Админ: глобальная статистика, экспорт в CSV/Excel.

⚠️ Важное правило: уведомления MLM не содержат персональных данных — только агрегированные факты (например: «У вас появился новый партнёр на 2-й линии»).
